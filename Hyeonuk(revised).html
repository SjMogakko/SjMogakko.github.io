  • 이중포인터

    포인터도 변수이므로 어떠한 포인터의 메모리주소를 다른포인터에 저장할수있다

      int **Ptr2;
  int *Ptr1;
  int n=10;
  Ptr1=&n;//변수 n의 메모리주소 Ptr1에저장
  Ptr2=&Ptr1;//변수 Ptr의 메모리주소 Ptr2에저장
  printf("%d\n",**Ptr2);//역참조 2번하므로 n값인 10
  printf("%d\n",*Ptr1);//위값과 같음
  printf("%p\n",*Ptr2);//역참조 1번하므로 변수 n의 메모리주소값인 Ptr1
  printf("%p\n",Ptr1);//위값과 같음

    • <stdlib.h> 에있는함수인 malloc,free를 이용해서 포인터에 원하는만큼 메모리 할당하기

    • Allocating memory to pointer

    • int *list;
list=malloc(sizeof(int)*8);//memory allocate의 줄임말인 malloc,int형은 4바이트 * 8개, list는 8개의 int자료형을 저장할수있는 리스트와 같다
free(list)//메모리를 할당해주면 free함수를 이용하여 메모리해제를 해주어야 한다

    • Array - pointer(메모리 할당을 이용하여 입력받은 변수 만큼의 배열만들기)(1~n)

      • int *list,n;
scanf("%d",&n);
list=malloc(sizeof(int)*n);
for(int i=0;i<n;i++)list[i]=i+1;//list[0]에1,list[1]에2,......list[n-1]에n저장
free(list)//동적으로 할당한 메모리 해제

    • 이중 포인터를 사용하여 단위행렬을 입력받은 수만큼의 row by column 으로 만들기

      • int **I_matrix;
int row_col;
scanf("%d",&row_col);//행==열 갯수을 입력받는다
I_matrix=malloc(sizeof(int*)*n);//row와 col의길이가 다를떄는 row갯수만큼 먼저 메모리할당(행의갯수),여기서 int*인이유는 int형의 1중 포인터(pointer to int)를 이중포인터에 저장하는것이기 떄문이다.(사실 잘모르겠다...)
for(int i=0;i<row_col;i++) I_matrix[i]=malloc(sizeof(int)*row_col);//col갯수만큼 각 row에 메모리할당(열의갯수)
for(int i=0;i<row_col;i++)
{
	for(int j=0;j<row_col;j++)
	{
		if(i==j) I_matrix[i][j]=1;//단위행렬을 만들어줌
		else I_matrix[i][j]==0;
	}
}
for(int i=0;i<row_col;i++)
{
    for(int j=0;j<row_col;j++)
    {
        printf("%d ",I_matrix[i][j]);//단위행렬 출력
    }
    printf("\n");
}
for(int i=0;i<row_col;i++) free(I_matrix[i]);//세로 길이만큼 반복 free
free(I_matrix);

  • Strings(문자열)

    Declaring String

    • Using pointer

      문자열이 있는 메모리주소를 포인터에 할당함(**읽기전용!!**)

      char*c1="Hello World";
//Print
printf("%s",c1);

      포인터를 사용한 문자열은 읽기전용이므로 요소를 바꿀수없음

      char *c1="Hello world";
c1[0]=G;
printf("%s",c1);

      **위와같은 코드는 오류를 불러일으킨다**

    • Using array

      char형 변수를 선언하고 그뒤에 알파뱃갯수+띄어쓰기갯수+1(마지막 NULL)보다 큰 수를 변수뒤에 배열선언하듯이 써주면된다.

      char c1[12]="Hello world";
//Print
printf("%s",c1);

      배열을 사용한 문자열은 포인터와다르게 문자열의 요소를 바꿀수있음

      char c1[12]="Hello world";
c1[0]='A';
printf("%s",c1);//Aello world

      또한 이미 선언된 배열을 사용한 문자열에 새로운 문자열을 한번에 추가불가능함

      char c1[12];
c1="Hello World";
printf("%s",c1);

      **위와같은 코드는 마찬가지로 오류를 불러일으킨다**
