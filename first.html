알고리즘

탐색 알고리즘

  1. 선형탐색

    선형탐색은 리스트의 첫 값부터 마지막 값까지 순차적으로 회전하면서 특정 요소를 찾아내는 알고리즘이다.

    def linear_search(element, some_list):
    for i in range(len(some_list)):
        if some_list[i] == element : return i
    return None

  2. 이진 탐색

    이진탐색은 정렬된 리스트를 기준으로, 중앙값을 확인하고 찾으려는 요소가 왼쪽에 있는지, 오른쪽에 있는지를 판단하여 탐색한다. 정렬된 리스트에서만 사용할 수 있으나. 대체적으로 선형탐색보다 효율적인 시간 복잡도를 보인다.

    def binary_search(element, some_list):
    length = len(some_list)
    indexs = range(length)
    while True:
        x = len(indexs)
        if x == 0 : return None
        idx = indexs[x//2]
        if some_list[idx] == element : return idx
        elif element < some_list[idx] : indexs = indexs[:x//2]
        else : indexs = indexs[x//2+1:]
구분 선형탐색 이분탐색
시간복잡도 O(n) O(lg n)
정렬 유무 X O

정렬 알고리즘

  1. 선택정렬

    리스트 전체를 순회하여 최소값(또는 최댓값)을 찾아낸 뒤에 이를 맨 끝으로 보내는 작업을 반복하여 정렬된 리스트를 만들어내는 알고리즘이다.

   def sel_sort(a):
     n = len(a)
     for i in range(0, n - 1):
       min_idx = i
       for j in range(i + 1, n):
         if a[j] < a[min_idx]:
           min_idx = j
       a[i], a[min_idx] = a[min_idx], a[i]

  1. 삽입 정렬

    각 인덱스의 값들이 정렬된 값들과 비교해 어디에 삽입될지를 정함으로써 정렬을 해나가는 알고리즘이다.

   def insertion_sort(my_list):
       for i in range(len(my_list)):
           key = my_list[i]
           j = i - 1
           while j >= 0 and my_list[j] > key:
               my_list[j + 1] = my_list[j]
               j = j - 1
           my_list[j + 1] = key

  1. 병합정렬

    리스트를 모두 쪼개어냈다가 다시 합치는 순간에 정렬을 하는 알고리즘으로, 분할정복 방식의 정렬 알고리즘이다.

    def merge(list1, list2):
    answer = []
    while True :
        if list1 == [] : return answer+list2
        if list2 == [] : return answer+list1
        if list1[0] <= list2[0]:
            answer.append(list1.pop(0))
        else :
            answer.append(list2.pop(0))

def merge_sort(my_list):
    length = len(my_list)
    if length < 2 : return my_list
    list1 = my_list[:length//2]
    list2 = my_list[length//2:]
    return merge(merge_sort(list1), merge_sort(list2))

  2. 퀵 정렬

    분할정복 방식의 정렬 알고리즘, 특정 값을 기준으로 작은 값을 왼쪽에, 큰 값을 오른쪽에 두는 과정을 반복함으로써 정렬한다.

   def swap_elements(my_list, index1, index2):
       temp = my_list[index1]
       my_list[index1] = my_list[index2]
       my_list[index2] = temp
       return my_list

   def partition(my_list, start, end):
       b = start
       pivot = my_list[end]
       for i in range(start,end):
           if my_list[i] < pivot :
               b += 1
               my_list = swap_elements(my_list, b-1, i)
       my_list = swap_elements(my_list,b,end)
       return b

    def quicksort(my_list, start, end):
    if start >= end : return
    else :
        idx = partition(my_list,start,end)

        quicksort(my_list,start,idx-1)
        quicksort(my_list,idx+1,end)

알고리즘 접근법

Brute Force

가능한 모든 경우의 수를 구하고, 해당 경우들 중 가장 적합한 해를 찾아내는 것을 의미한다. 직관적으로 풀수 있지만, 그만큼 시간복잡도나 공간복잡도가 비효율 적일 가능성이 크다.

예제

카드 두 뭉치가 있습니다.

왼쪽 뭉치에서 카드를 하나 뽑고 오른쪽 뭉치에서 카드를 하나 뽑아서, 두 수의 곱이 가장 크게 만들고 싶은데요. 어떻게 하면 가장 큰 곱을 구할 수 있을까요?

함수 `max_product`는 리스트 `left_cards`와 리스트 `right_cards`를 파라미터로 받습니다.

`left_cards`는 왼쪽 카드 뭉치의 숫자들, `right_cards`는 오른쪽 카드 뭉치 숫자들이 담겨 있고, `max_product`는 `left_cards`에서 카드 하나와 `right_cards`에서 카드 하나를 뽑아서 곱했을 때 그 값이 최대가 되는 값을 리턴합니다.

코드

def max_product(left_cards, right_cards):
    max = left_cards[0] * right_cards[0]
    for left in left_cards:
        for right in right_cards:
            if max < left * right : max = left*right
    return max

Divide and Conquer(분할 정복)

​ 한 문제를 부분 문제로 나누고, 그 부분문제들의 답을 이용하여 본래 문제의 답을 구하는 풀이방식이다. 보통 재귀함수의 방식을 통해 구현된다.

예제

함수 `consecutive_sum`은 두 개의 정수 인풋 `start`와 `end`를 받고, `start`부터 `end`까지의 합을 리턴합니다. `end`는 `start`보다 크다고 가정합니다.

코드

def consecutive_sum(start, end):
    if start == end : return end
    else :
        return consecutive_sum(start,(end+start)//2) + consecutive_sum((end+start)//2 + 1 , end)

병합 정렬과 퀵 정렬 역시 분할 정복의 일종이다.

Dynamic Programming(동적 할당)

​ 알고리즘 내에서 이미 한번이상 했던 작업을 반복하지 않도록 작성하는 방법이다. Tabulation과 Memorization으로 구분되며, 최적부분구조와 중복되는 부분문제 조건이 성립할때 사용하는 방식이다.

  1. Tabulation

    보통 dp라고 명명하는 리스트를 만들어, 순차적으로 리스트 안 값들을 채워나가는 방식이다. 리스트 내에 연산 결과가 저장되므로, 중복연산을 하지 않게 된다.

    예제(피보나치수열)
    def fib_tab(n):
    # 이미 계산된 피보나치 수를 담는 리스트
    fib_table = [0, 1, 1]

    # n번째 피보나치 수까지 리스트를 하나씩 채워 나간다
    for i in range(3, n + 1):
        fib_table.append(fib_table[i - 1] + fib_table[i - 2])

    # 피보나치 n번째 수를 리턴한다
    return fib_table[n]

  2. Memorize

    한번 작업한 연산들의 결과를 cache(캐시)에 저장하여 다음 연산부터는 불러오는 방식이다.

    예제(피보나치수열)
    def fib_memo(n, cache={}):
    # base case
    if n < 3:
        return 1

    # 이미 n번째 피보나치를 계산했으면:
    # 저장된 값을 바로 리턴한다
    if n in cache:
        return cache[n]

    # 아직 n번째 피보나치 수를 계산하지 않았으면:
    # 계산을 한 후 cache에 저장
    cache[n] = fib_memo(n - 1, cache) + fib_memo(n - 2, cache)

Greedy(탐욕법)

​ 최적의 해가 아닌 적당한 정답만 있어도 될때 사용한다. 필수 조건은 존재하지 않지만, 부분최적구조와 탐욕적 선택속성을 만족할 경우 이 알고리즘으로 최적해 를 알아낼 수 있다.

예제
	함수 `max_product`는 한 사람당 카드를 하나씩 뽑아서 모두 곱했을 때 가능한 최대 곱을 리턴합니다.

	예시 : test_cards = [[1, 2, 3], [4, 6, 1], [8, 2, 4], [3, 2, 5], [5, 2, 3], [3, 2, 1]]
코드
def max_product(card_lists):
	answer = 1
	for cards in card_lists:
   		answer *= max(cards)
	return answer

상속

상속은 자식 클래스가 부모 클래스의 특성과 기능을 그대로 물려받는 것을 말한다. 기능의 일부분을 변경해야 할 경우 자식 클래스에서 상속받은 그 기능만을 수정해서 다시 정의하게 되는데, 이러한 작업을 ‘오버라이딩(Overriding)’이라고 한다. 상속은 캡슐화를 유지하면서도 클래스의 재사용이 용이하도록 해 준다.

def counting(Square):
    """
    basecase : 정사각형 요소가 모두 한개로 통일 되어있다면, 그 수에 따라 결과값을 리턴한다
    1. 정사각형을 4등분으로 쪼갠다
    2. 각각의 쪼갠 사각형에 counting을 적용한다
    3. 카운팅으로 모인 수들을 더해서 return 한다
    """
    No1 = False
    No0 = False
    for line in Square:
        if 1 in line : No1 = True
        if 0 in line : No0 = True

    if No1 and not No0 : return [0,1]
    if not No1 and No0 : return [1,0]
    #여기까지 basecase

    length = len(Square)
    x = length // 2

    Square1 = []
    Square2 = []
    Square3 = []
    Square4 = []

    for i in range(x):
        Square1.append(Square[i][:x])
        Square2.append(Square[i][x:])

    for i in range(x, length):
        Square3.append(Square[i][:x])
        Square4.append(Square[i][x:])
    #쪼개기 완료

    count1, count2, count3, count4 = counting(Square1), counting(Square2), counting(Square3), counting(Square4)

    num_of_Zero = count1[0] + count2[0] + count3[0] + count4[0]
    num_of_One = count1[1] + count2[1] + count3[1] + count4[1]

    return([num_of_Zero, num_of_One])


N = int(input())

shape = []
for i in range(N):
    shape.append(list(map(int, input().split())))

answer = counting(shape)
print(answer[0])
print(answer[1])

def idx(k):
    #basecase : k == 1
    if k == 1 :
        return 0
    else:
        n = 1
        while True:
            if 2**(n-1) < k <= 2**n : break
            n += 1
        index = k-(2**(n-1))
        #print(index)
        return 1-idx(index)
