모각코 첫주 활동 정리

월요일

 월요일은 모각코 첫 활동일로 활동 방법을 소개하고 공부 목표를 세웠다. 나는 다른 사람과 함께 알고리즘을 공부하기로 하였다. 공부하고자 하는 알고리즘을 정하여 몇개의 문제를 뽑아 각자 문제를 풀어오고 모각코 활동 시간동안 서로의 풀이를 보고 개선 방안 등을 이야기하기로 하였다.

화요일

Dynamic Programming

BOJ 12865

  1. 문제 내용
    • 0-1 knapsack 문제
  2. 접근
    • 각 물건을 골랐을 때와 안 골랐을 때를 나눠 모든 경우의 수를 구하려고 시도함
    • 물건의 개수가 n일 때 2^n 가지의 경우가 나와 제한 시간 안에 구할 수 없음
  3. 찾은 풀이 방법
    • 각 물건을 넣을 때와 안 넣었을 때 최선의 선택을 찾음
    • 이전에 구한 최선의 선택을 이용하여 이후 선택에서도 최선의 선택을 쉽게 구할 수 있게 함 (타블레이션)
    • 시간이 8000ms로 줄었지만 메모리가 약 220MB가 사용됨
    • 추가 조사를 통하여 일정한 크기의 범위의 데이터를 처리할 경우 바로 직전 데이터를 남겨두어 이후 계산을 빠르게 하도록 하는 방법인 슬라이딩 윈도우 기법을 알게 됨
    • 타블레이션을 통하여 문제를 풀 때 현재 행을 계산하기 위해서 바로 직전 행만 필요하다는 점을 이용하여 두개의 행만 메모리에 남겨두는 방법으로 슬라이딩 윈도우를 적용함
    • 결과적으로 메모리 사용량이 36MB로 줄었고 메모리 접근 시간을 줄여 실행 속도 또한 6600ms로 빨라짐
  4. 추가 공부 내용
    • 슬라이딩 윈도우
    • 투 포인트 알고리즘

BOJ 1106

  1. 문제 내용
    • 0-1 knapsack 문제에서 목표 만족도를 달성하기 위한 최소 비용을 요구하는 문제
  2. 접근
    • 12865번 문제에 적용한 내용을 이용하여 K를 하나씩 늘려가며 목표값이 나올 때의 K를 구하고자 함
    • 적용에 실패했고, 이후에 다시 풀어볼 예정
